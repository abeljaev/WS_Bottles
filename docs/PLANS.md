# Планы развития

## Краткосрочные (1-2 недели)

### 1. Тестирование
- Unit тесты для core модулей
- Integration тесты для WebSocket коммуникации
- Mock для PLC и камеры

### 2. Отказоустойчивость
- Retry механизмы для Modbus
- Reconnect логика для WebSocket
- Graceful shutdown

## Среднесрочные (1 месяц)

### 3. Backend интеграция
- REST/WebSocket API для внешнего бэкенда
- Аутентификация клиентов
- Очередь команд

### 4. Мониторинг
- Prometheus метрики
- Health check endpoints
- Alerting

## Долгосрочные

### 5. Масштабирование
- Поддержка нескольких камер
- Кластеризация
- Remote management
